Of Acid and Steel

My largest (And still undergoing) personal project, this is a Turn-Based, Fantasy RPG created using RPG Maker. Although the game itself is created mainly using the events in the engine, I've had to develop several script-based events and use a few add-ons as well! The coding is done in JavaScript, and all of the work in this game is my own- Art, Character development, Combat, and More!

I have a personal goal to make this game all on my own, no matter how long it takes- And I plan on getting into every part of it, from music to animation. During the fist few months of working on the beginning, I've gotten over 50 downloads in it!

Ryophi Species Mod (Starbound)

I've created a unique species mod for the game Starbound (A Terraria-Style game in a Sci-Fi setting), snake-like humanoids called the Ryophi. The mod for Starbound is public on the Steam Workshop, and has over 400 users! Starbound, and by extension the mod, is written in JavaScript, and uses JSON objects.

While some of the mod's work was in sprites, I've also programmed the species to be unique- A larger hitbox, unique particles around their models, and unique text for item descriptions! The Ryophi are also planned to be more fleshed-out lore-wise in the future, as I've created a page on Toyhouse for them.
